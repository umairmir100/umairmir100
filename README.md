### 🚀 Featured Projects

#### 🖐️ [Finger Counter](https://github.com/umairmir100/finger-counter)
Real-time hand-tracking app that counts how many fingers are raised using **MediaPipe** and **OpenCV**.
- Tracks hand landmarks frame-by-frame
- Classifies finger states (up/down) using landmark geometry
- Runs live on webcam feed with no extra hardware

`MediaPipe` `OpenCV` `Python`

#### 🛒 [RetailEye](https://github.com/umairmir100/retail-eye)
A people-detection and analytics pipeline for retail spaces — works on images, video files, and live webcam feeds.
- Detects and counts people per defined zone
- Logs detection events with timestamps for downstream analytics
- Built with deployment in mind using **DeepStream** for real-time inference

`PyTorch` `OpenCV` `DeepStream`
